title: "ロール"
description: "ロールでダッシュボードへのアクセスを制御します。"
---

<Info>
  RBAC 機能は[Enterprise プラン](https://mintlify.com/pricing?ref=rbac)で利用可能です。
</Info>

Mintlify では、ダッシュボードのアクセスレベルとして Editor と Admin の 2 種類を提供しています。

以下は Admin ロールのみが実行できるアクションです:

|                         | Editor | Admin |
| ----------------------- | :----: | :---: |
| ユーザーロールの更新           |   ❌   |  ✅   |
| ユーザーの削除                |   ❌   |  ✅   |
| 管理者ユーザーの招待          |   ❌   |  ✅   |
| 請求の管理・更新              |   ❌   |  ✅   |
| カスタムドメインの更新         |   ❌   |  ✅   |
| Git ソースの更新             |   ❌   |  ✅   |
| 組織の削除                   |   ❌   |  ✅   |

その他のダッシュボード上のアクションは、どちらのロールでも利用可能です。

管理者は必要な人数だけ招待できますが、必要なユーザーに限定することをおすすめします。
