アコーディオンは、ユーザーがコンテンツセクションを展開・折りたためるようにします。段階的な情報開示や情報整理のためにアコーディオンを使用してください。


<div id="single-accordion">
  ## 単一のアコーディオン
</div>

<Accordion title="I am an Accordion.">
  ここには、コードなどの他のコンポーネントを含む任意のコンテンツを入れられます：
   ```java HelloWorld.java
    class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
    ```
</Accordion>

<div id="accordion-groups">
  ## アコーディオングループ
</div>

関連するアコーディオンは `<AccordionGroup>` を使ってまとめます。これにより、各アコーディオンを個別に展開・折りたたみできる一体的なセクションを作成できます。

<AccordionGroup>
  <Accordion title="Getting started">
    アコーディオン内に他のコンポーネントを入れられます。

    ```java HelloWorld.java
    class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
    ```
  </Accordion>

  <Accordion title="Advanced features" icon="bot">
    アイコンを追加すると、視覚的な区別がつきやすくなり、一覧性も向上します。
  </Accordion>
  
  <Accordion title="Troubleshooting">
    関連コンテンツをグループとして整理できます。
  </Accordion>
</AccordionGroup>

<div id="properties">
  ## プロパティ
</div>

<ResponseField name="title" type="string" required>
  Accordion のプレビューに表示されるタイトル。
</ResponseField>

<ResponseField name="description" type="string">
  Accordion のプレビューでタイトルの下に表示される詳細。
</ResponseField>

<ResponseField name="defaultOpen" type="boolean" default="false">
  Accordion を既定で開いておくかどうか。
</ResponseField>
