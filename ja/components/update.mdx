統一されたフォーマットで変更履歴、バージョン更新、リリースノートを表示するには、`Update` コンポーネントを使用します。

<Update label="例" description="v0.1.1">
  ## 例のアップデート

  スクリーンショット、コードスニペット、変更点のリストなど、ここに自由に追加できます。

  <Frame>
    <img className="block" src="https://mintlify-assets.b-cdn.net/hero-light.png" alt="" />
  </Frame>

  ### 機能

  * レスポンシブデザイン
  * 各アップデートへのアンカー
  * 各アップデートの RSS フィードエントリを生成
</Update>

<div id="how-to-use">
  ## 使い方
</div>

```mdx Update example
<Update label="2024-10-11" description="v0.1.0" tags={["Mintlify"]}>
  これはラベル、説明、タグを含む更新です。
</Update>
```

複数の `Update` コンポーネントを使って、[変更履歴](/ja/guides/changelogs)を作成します。

<div id="props">
  ## Props
</div>

<ResponseField name="label" type="string" required>
  更新のラベル。更新の左側に表示され、アンカーリンクを生成します。ラベルは一意である必要があります。
</ResponseField>

<ResponseField name="tags" type="string[]">
  更新のタグ。右側のパネルでフィルターとして表示されます。
</ResponseField>

<ResponseField name="description" type="string">
  更新の説明。ラベルとタグの下に表示されます。
</ResponseField>
