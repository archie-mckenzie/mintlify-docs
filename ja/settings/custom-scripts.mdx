CSSでHTML要素をスタイル指定したり、カスタムCSSやJavaScriptを追加して、ドキュメントの見た目や体験を細かくカスタマイズできます。

<div id="styling-with-tailwind-css">
  ## Tailwind CSSによるスタイリング
</div>

HTML要素のスタイル設定にTailwind CSS v3を使用します。レイアウト、間隔、色などの視覚的プロパティを制御できます。よく使われるクラスは次のとおりです:

- `w-full` - 全幅
- `aspect-video` - 16:9 のアスペクト比
- `rounded-xl` - 大きめの角丸
- `block`, `hidden` - 表示の切り替え
- `dark:hidden`, `dark:block` - ダークモードでの表示切り替え

Tailwind CSSの任意値はサポートされていません。カスタム値には `style` プロップを使用してください。

```html
<img style={{ width: '350px', margin: '12px auto' }} src="/path/image.jpg" />
```

<div id="custom-css">
  ## カスタムCSS
</div>

CSSファイルをリポジトリに追加すると、そこで定義したクラス名が適用され、すべての`MDX`ファイルで利用できるようになります。

<div id="adding-stylecss">
  ### `style.css` の追加
</div>

例えば、ナビバーとフッターのスタイルをカスタマイズするために、次の `style.css` ファイルを追加できます。

```css
#navbar {
  background: "#fffff2";
  padding: 1rem;
}

footer {
  margin-top: 2rem;
}
```

<div id="custom-javascript">
  ## カスタム JavaScript
</div>

カスタムJSを使うと、任意の実行コードをグローバルに追加できます。これは、すべてのページに JS コードを含む `<script>` タグを挿入するのと同等です。

<div id="adding-custom-javascript">
  ### カスタムJavaScriptの追加
</div>

ドキュメントのコンテンツディレクトリ内にある `.js` ファイルは、すべてのドキュメントページで読み込まれます。例えば、以下の `ga.js` ファイルを追加すると、ドキュメント全体で [Google Analytics](https://marketingplatform.google.com/about/analytics) を有効化できます。

```js
window.dataLayer = window.dataLayer || [];
function gtag() {
  dataLayer.push(arguments);
}
gtag('js', new Date());

gtag('config', 'TAG_ID');
```

<Warning>
  セキュリティ上の脆弱性を生じさせないよう、十分注意して使用してください。
</Warning>
