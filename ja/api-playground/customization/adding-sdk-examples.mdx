---
title: "SDK のサンプルを追加する"
description: "API エンドポイントごとに言語別のコードサンプルを表示して、開発者に SDK の使い方を示します"
---

ユーザーがネットワークリクエストを直接送るのではなく SDK を使って API とやり取りする場合は、`x-codeSamples` 拡張を使って OpenAPI ドキュメントにコードサンプルを追加し、OpenAPI ページに表示できます。

このプロパティは任意のリクエストメソッドに追加でき、以下のスキーマを持ちます。

<ParamField body="lang" type="string" required>
  コードサンプルの言語。
</ParamField>

<ParamField body="label" type="string">
  サンプルのラベル。単一のエンドポイントに複数の例を提供する際に便利です。
</ParamField>

<ParamField body="source" type="string" required>
  サンプルのソースコード。
</ParamField>

以下は、Bash の CLI ツールと JavaScript の SDK の両方を備えた植物トラッキングアプリのコードサンプル例です。

```yaml
paths:
  /plants:
    get:
      # ...
      x-codeSamples:
        - lang: bash
          label: List all unwatered plants
          source: |
            planter list -u
        - lang: javascript
          label: List all unwatered plants
          source: |
            const planter = require('planter');
            planter.list({ unwatered: true });
        - lang: bash
          label: List all potted plants
          source: |
            planter list -p
        - lang: javascript
          label: List all potted plants
          source: |
            const planter = require('planter');
            planter.list({ potted: true });
```
