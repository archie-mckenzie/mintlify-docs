<div id="enabling-data-privacy-integrations">
  ## データプライバシー統合の有効化
</div>

ドキュメントにデータプライバシープラットフォームを追加できます。`docs.json` に `integrations` フィールドを追加し、該当するスクリプトを設定してください。

```json
  "integrations": {
    "osano": "SOURCE"
  }
```

<div id="cookie-consent-and-disabling-telemetry">
  ## Cookie の同意とテレメトリーの無効化
</div>

GDPR 準拠のために、ユーザーがすでに Cookie に同意しているかを確認する必要がある場合は、`cookies` の下にローカルストレージのキーと値を指定できます。

```json
  "integrations": {
    "cookies": {
      "key": "LOCAL STORAGE KEY",
      "value": "LOCAL STORAGE VALUE"
    }
  }
```

これらの値が設定されている場合、ローカルストレージを参照してユーザーの Cookie 同意有無を確認します。未同意の場合は、テレメトリーを無効化します。

すべてのユーザーに対してテレメトリーを無効化したい場合は、`docs.json` ファイルに次を追加します。

```json
  "integrations": {
    "telemetry": {
      "enabled": false
    }
  }
```
