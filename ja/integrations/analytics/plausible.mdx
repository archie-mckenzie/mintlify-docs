---
title: "Plausible"
---

サイトのドメインを `docs.json` に追加して、Plausible に分析データを送信します。

<Info>

    ドメインまたはサーバーには `https://` を含めないでください。

</Info>

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "plausible": {
        "domain": "required",
        "server": "optional"
    }
}
```

```json Example
"integrations": {
    "plausible": {
        "domain": "docs.domain.com"
    }
}
```

</CodeGroup>
