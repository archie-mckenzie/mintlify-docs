---
title: "Pirsch"
---

分析を Pirsch に送信するには、`docs.json` に次を追加します。

サイト ID は Settings > Developer > Identification Code から取得できます。

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "pirsch": {
        "id": "required"
    }
}
```

```json Example
"integrations": {
    "pirsch": {
        "id": "8Kw7OKxBfswOjnKGZa7P9Day8JmVYwTp"
    }
}
```

</CodeGroup>
