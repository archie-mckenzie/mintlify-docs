---
title: "Google Tag Manager"
---

`docs.json` ファイルにタグ ID を追加すると、すべてのページに Google Tag Manager のスクリプトを自動的に挿入します。

必要に応じて、Cookie 同意バナーの設定は Google Tag Manager 側で行ってください。

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "gtm": {
        "tagId": "required"
    }
}
```

```json Example
"integrations": {
    "gtm": {
        "tagId": "GTM-MGBL4PW"
    }
}
```

</CodeGroup>
