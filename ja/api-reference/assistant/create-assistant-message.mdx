<div id="rate-limits">
  ## レート制限
</div>

Assistant API には次の制限があります：

- キーごとに月あたり 10,000 回の利用
- Mintlify 組織ごとに1時間あたり 10,000 リクエスト
- IP ごとに1日あたり 10,000 リクエスト

<div id="suggested-usage">
  ## 推奨される使い方
</div>

最良の結果を得るには、リクエスト送信とレスポンス処理に [ai-sdk の useChat フック](https://ai-sdk.dev/docs/reference/ai-sdk-ui/use-chat#usechat) を使用してください。

フックに渡すオプションの `body` フィールドで `fp`、`threadId`、`filter` を設定できます。
