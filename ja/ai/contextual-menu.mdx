コンテキストメニューは、AI 最適化済みコンテンツや主要な AI ツールとのダイレクト連携に素早くアクセスできます。ユーザーが任意のページでコンテキストメニューを開くと、AI ツール向けのコンテキストとしてコンテンツをコピーしたり、ChatGPT、Claude、Perplexity、または任意のカスタムツールで、ドキュメントがすでにコンテキストとして読み込まれた会話を開始できます。


<div id="menu-options">
  ## メニューオプション
</div>

コンテキストメニューには、設定に識別子を追加することで有効化できる、あらかじめ用意されたオプションが含まれています。

| オプション | 識別子 | 説明 |
|:--------|:------------|:-------------|
| **Copy page** | `copy` | 現在のページを Markdown としてコピーし、AI ツールにコンテキストとして貼り付けられるようにします |
| **View as Markdown** | `view` | 現在のページを Markdown として開きます |
| **Open in ChatGPT** | `chatgpt` | 現在のページをコンテキストとする ChatGPT の会話を作成します |
| **Open in Claude** | `claude` | 現在のページをコンテキストとする Claude の会話を作成します |
| **Open in Perplexity** | `perplexity` | 現在のページをコンテキストとする Perplexity の会話を作成します |
| **Copy MCP server URL** | `mcp` | MCP サーバーの URL をクリップボードにコピーします |
| **Connect to Cursor** | `cursor` | ホスト中の MCP サーバーを Cursor にインストールします |
| **Connect to VS Code** | `vscode` | ホスト中の MCP サーバーを VS Code にインストールします |

<div id="enabling-the-contextual-menu">
  ## コンテキストメニューを有効にする
</div>

`docs.json` ファイルに `contextual` フィールドを追加し、含めるオプションを指定します。

```json
{
 "contextual": {
   "options": [
     "copy",
     "view",
     "chatgpt",
     "claude",
     "perplexity",
     "mcp",
     "cursor",
     "vscode"
   ]
 }
}
```

<div id="adding-custom-options">
  ## カスタムオプションの追加
</div>

`options` 配列にオブジェクトを追加して、コンテキストメニューにカスタムオプションを作成します。各カスタムオプションには次のプロパティが必要です:

<ResponseField name="title" type="string" required>
  オプションのタイトル。
</ResponseField>

<ResponseField name="description" type="string" required>
  オプションの説明。コンテキストメニューを展開すると、タイトルの下に表示されます。
</ResponseField>

<IconsRequired />

<ResponseField name="href" type="string | object" required>
  オプションの href。単純なリンクには文字列を、クエリパラメータを含む動的リンクにはオブジェクトを使用します。
  
  <Expandable title="href object">
    <ResponseField name="base" type="string" required>
      オプションのベースURL。
    </ResponseField>

    <ResponseField name="query" type="object" required>
      オプションのクエリパラメータ。

      <Expandable title="query object">
        <ResponseField name="key" type="string" required>
          クエリパラメータのキー。
        </ResponseField>

        <ResponseField name="value" type="string" required>
          クエリパラメータの値。次のプレースホルダーは対応する値に置き換えられます:
          - `$page` 現在のページのコンテンツ（Markdown）
          - `$path` 現在のページパス
          - `$mcp` ホストされている MCP サーバーのURL
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

カスタムオプションの例:

```json {9-14} wrap
{
    "contextual": {
        "options": [
            "copy",
            "view",
            "chatgpt",
            "claude",
            "perplexity",
            {
                "title": "Request a feature",
                "description": "Join the discussion on GitHub to request a new feature",
                "icon": "plus",
                "href": "https://github.com/orgs/mintlify/discussions/categories/feature-requests"
            }
        ]
    }
}
```
