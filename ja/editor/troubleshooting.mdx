ウェブエディターの利用時によく発生する問題の解決策を紹介します。

<AccordionGroup>
  <Accordion title="公開後に変更が反映されない">
    **考えられる原因:**

    * デプロイがまだ進行中
    * ブラウザのキャッシュが原因
    * ビルドまたはデプロイのエラー

    **解決策:**

    1. ダッシュボードでデプロイのステータスを確認してください。
    2. ブラウザをハードリロードします（<kbd>Ctrl</kbd> + <kbd>F5</kbd> または <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>R</kbd>）。
    3. ブラウザのキャッシュをクリアしてください。
  </Accordion>

  <Accordion title="公開時の権限エラー">
    **考えられる原因:**

    * Git リポジトリの権限不足
    * 利用中の Git プロバイダでの認証問題

    **解決策:**

    1. リポジトリへの適切なアクセス権があるか確認してください。
    2. Git 連携が正しく設定されているか確認してください。
    3. [Editor Permissions](/ja/advanced/dashboard/permissions) のドキュメントを参照してください。
  </Accordion>

  <Accordion title="エディターが読み込めない／読み込みが遅い">
    **考えられる原因:**

    * ネットワーク接続の問題
    * ドキュメントリポジトリが大きい

    **解決策:**

    1. インターネット接続を確認してください。
    2. ページを再読み込みしてください。
    3. 解消しない場合はサポートにお問い合わせください。
  </Accordion>

  <Accordion title="ファイルが読み込めない、またはエラーが表示される">
    **考えられる原因:**

    * ファイル内の MDX 構文エラー
    * ファイルの欠落または破損
    * 大きなファイルサイズによるタイムアウト

    **解決策:**

    1. MDX の構文エラーがないかファイルを確認してください。
    2. リポジトリに対象ファイルが存在するか確認してください。
  </Accordion>
</AccordionGroup>
