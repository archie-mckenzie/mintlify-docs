<div id="lists">
  ## リスト
</div>

リストは公式の[Markdown構文](https://www.markdownguide.org/basic-syntax/#lists-1)に準拠します。

<div id="ordered-list">
  ### 番号付きリスト
</div>

番号付きリストを作成するには、各項目の先頭に数字とピリオドを付けます。

1. 1番目の項目
2. 2番目の項目
3. 3番目の項目
4. 4番目の項目

```mdx
1. First item
2. Second item
3. Third item
4. Fourth item
```

<div id="unordered-list">
  ### 箇条書き
</div>

箇条書きを作成するには、各項目の前にハイフン（`-`）、アスタリスク（`*`）、またはプラス（`+`）を付けます。

- 1つ目の項目
- 2つ目の項目
- 3つ目の項目
- 4つ目の項目

```mdx
- First item
- Second item
- Third item
- Fourth item
```

<div id="nested-list">
  ### 入れ子リスト
</div>

リスト項目をインデントして入れ子にします。

- 1つ目の項目
- 2つ目の項目
  - 追加の項目
  - 追加の項目
- 3つ目の項目

```mdx
- First item
- Second item
  - Additional item
  - Additional item
- Third item
```

<div id="tables">
  ## テーブル
</div>

テーブルは公式の[Markdown構文](https://www.markdownguide.org/extended-syntax/#tables)に準拠します。

テーブルを追加するには、3つ以上のハイフン（`---`）で各列のヘッダーを作成し、パイプ（`|`）で各列を区切ります。互換性のため、行の両端にもパイプを追加してください。

| プロパティ | 説明                                   |
| ---------- | -------------------------------------- |
| 名前       | ユーザーの氏名                          |
| 年齢       | 申告された年齢                          |
| 参加       | ユーザーがコミュニティに参加しているか |

```mdx
| Property | Description                           |
| -------- | ------------------------------------- |
| Name     | Full name of user                     |
| Age      | Reported age                          |
| Joined   | Whether the user joined the community |
```
