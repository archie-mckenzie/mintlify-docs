<Info>
  Vorschau-Deployments sind in den [Pro- und Enterprise-Plänen](https://mintlify.com/pricing?ref=preview-deployments) verfügbar.
</Info>

Mit Vorschau-Deployments sehen Sie, wie sich Änderungen an Ihrer Dokumentation auswirken, bevor sie in die Produktion übernommen werden. Jede Vorschau erzeugt eine teilbare URL, die sich automatisch aktualisiert, sobald Sie neue Änderungen pushen.

<div id="creating-preview-deployments">
  ## Vorschau-Bereitstellungen erstellen
</div>

Vorschau-Bereitstellungen werden automatisch über Pull Requests oder manuell über Ihr Dashboard erstellt.

<div id="automatic-previews">
  ### Automatische Vorschauen
</div>

Wenn Sie einen Pull Request eröffnen, fügt der Mintlify-Bot automatisch einen Link hinzu, über den Sie die Vorschau-Bereitstellung im Pull-Request einsehen können. Die Vorschau wird jedes Mal aktualisiert, wenn Sie neue Commits auf den Branch pushen.

<div id="manual-previews">
  ### Manuelle Vorschauen
</div>

Sie können eine Vorschau manuell für jeden Branch erstellen.

1. Gehen Sie zu Ihrem [Dashboard](https://dashboard.mintlify.com/).
2. Wählen Sie **Previews**.
3. Wählen Sie **Create preview**.
4. Geben Sie den Namen Ihres Branches ein.
5. Wählen Sie **Create deployment**.

<div id="redeploying-a-preview">
  ## Vorschau erneut bereitstellen
</div>

Stellen Sie eine Vorschau erneut bereit, um Inhalte zu aktualisieren oder nach einer fehlgeschlagenen Bereitstellung einen erneuten Versuch zu starten.

1. Wählen Sie die Vorschau in Ihrem [Dashboard](https://dashboard.mintlify.com/) aus.
2. Wählen Sie **Erneut bereitstellen**.

<div id="sharing-preview-deployments">
  ## Vorschau-Deployments teilen
</div>

Vorschau-Deployments können öffentlich zugänglich sein oder auf authentifizierte Benutzer beschränkt werden.

**Öffentlicher Zugriff**: Vorschau-URLs sind standardmäßig öffentlich sichtbar. Teilen Sie einen Vorschau-Link mit allen, die Ihre Änderungen prüfen müssen.

**Eingeschränkter Zugriff**: Aktivieren Sie die Authentifizierung für Vorschau-Deployments, um den Zugriff auf authentifizierte Mintlify-Benutzer zu beschränken. Weitere Informationen finden Sie unter [Einrichtung der Authentifizierung](/de/authentication-personalization/authentication-setup).
