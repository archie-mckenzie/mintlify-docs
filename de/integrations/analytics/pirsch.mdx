---
title: "Pirsch"
---

Füge Folgendes zu deiner `docs.json`-Datei hinzu, um Analysedaten an Pirsch zu senden.

Du kannst deine Site-ID unter Einstellungen \> Entwickler \> Identifikationscode abrufen.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "pirsch": {
        "id": "required"
    }
}
```

```json Example
"integrations": {
    "pirsch": {
        "id": "8Kw7OKxBfswOjnKGZa7P9Day8JmVYwTp"
    }
}
```

</CodeGroup>
