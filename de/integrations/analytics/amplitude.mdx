---
title: "Amplitude"
---

Fügen Sie Folgendes zu Ihrer `docs.json`-Datei hinzu, um Analytics an Amplitude zu senden.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "amplitude": {
        "apiKey": "required"
    }
}
```

```json Example
"integrations": {
    "amplitude": {
        "apiKey": "76bb138bf3fbf58186XXX00000"
    }
}
```

</CodeGroup>
