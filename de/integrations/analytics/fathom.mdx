---
title: "Fathom"
---

Fügen Sie Folgendes zu Ihrer `docs.json`-Datei hinzu, um Analysedaten an Fathom zu senden.

Die `siteId` finden Sie in Ihren Skripteinstellungen.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "fathom": {
        "siteId": "required"
    }
}
```

```json Example
"integrations": {
    "fathom": {
        "siteId": "YSVMSDAY"
    }
}
```

</CodeGroup>
