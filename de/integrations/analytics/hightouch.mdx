---
title: "Hightouch"
---

Fügen Sie Folgendes zu Ihrer `docs.json`-Datei hinzu, um Analytics an Hightouch zu senden.

<Info>

  Fügen Sie für den `apiHost` kein `https://` hinzu.

</Info>

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "hightouch": {
        "writeKey": "required"
        "apiHost": "optional"
    }
}
```

```json Example
"integrations": {
    "hightouch": {
        "writeKey": "9132c81do371p76sr11da0075469b54f77649c9a067dp0303p56q0q64n072336"
        "apiHost": "us-east-1.hightouch-events.com" # optional, defaults to `us-east-1.hightouch-events.com`
    }
}
```

</CodeGroup>
