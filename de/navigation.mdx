<div id="auto-navigation-on-expand">
  ### Automatische Navigation beim Erweitern
</div>

Verschiedene Themes haben unterschiedliches Standardverhalten beim Klicken auf Navigationsgruppen. Sie kÃķnnen dieses Verhalten mit der Option `drilldown` Ãžberschreiben:

```json
{
  "interaction": {
    "drilldown": true  // Erzwingt die Navigation zur ersten Seite, wenn ein Benutzer ein Dropdown erweitert
  }
}
```

```json
{
  "interaction": {
    "drilldown": false // Nie navigieren, nur erweitern/zusammenklappen
  }
}
```

- Auf `true` setzen, um beim Klicken auf Navigationsgruppen die automatische Navigation zur ersten Seite zu erzwingen
- Auf `false` setzen, um die Navigation zu verhindern und nur die Gruppe zu erweitern oder zusammenzuklappen
- Nicht setzen, um das Standardverhalten des Themes zu verwenden
