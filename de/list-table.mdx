<div id="lists">
  ## Listen
</div>

Listen folgen der offiziellen [Markdown-Syntax](https://www.markdownguide.org/basic-syntax/#lists-1).

<div id="ordered-list">
  ### Geordnete Liste
</div>

Um eine geordnete Liste zu erstellen, stellen Sie den Listeneinträgen Zahlen gefolgt von einem Punkt voran.

1. Erster Eintrag
2. Zweiter Eintrag
3. Dritter Eintrag
4. Vierter Eintrag

```mdx
1. First item
2. Second item
3. Third item
4. Fourth item
```

<div id="unordered-list">
  ### Ungeordnete Liste
</div>

Um eine ungeordnete Liste zu erstellen, stellen Sie den Listeneinträgen Bindestriche (`-`), Sternchen (`*`) oder Pluszeichen (`+`) voran.

- Erster Eintrag
- Zweiter Eintrag
- Dritter Eintrag
- Vierter Eintrag

```mdx
- First item
- Second item
- Third item
- Fourth item
```

<div id="nested-list">
  ### Verschachtelte Liste
</div>

Rücken Sie Listeneinträge ein, um sie zu verschachteln.

- Erster Eintrag
- Zweiter Eintrag
  - Zusätzlicher Eintrag
  - Zusätzlicher Eintrag
- Dritter Eintrag

```mdx
- First item
- Second item
  - Additional item
  - Additional item
- Third item
```

<div id="tables">
  ## Tabellen
</div>

Tabellen folgen der offiziellen [Markdown-Syntax](https://www.markdownguide.org/extended-syntax/#tables).

Um eine Tabelle hinzuzufügen, verwenden Sie drei oder mehr Bindestriche (`---`) für die Kopfzeile jeder Spalte und Pipes (`|`), um die Spalten zu trennen. Der Kompatibilität halber sollten Sie außerdem am Anfang und Ende jeder Zeile jeweils eine Pipe hinzufügen.

| Eigenschaft | Beschreibung                        |
| ----------- | ----------------------------------- |
| Name        | Vollständiger Name des Nutzers      |
| Alter       | Angegebenes Alter                   |
| Beigetreten | Ob der Nutzer der Community beigetreten ist |

```mdx
| Property | Description                           |
| -------- | ------------------------------------- |
| Name     | Full name of user                     |
| Age      | Reported age                          |
| Joined   | Whether the user joined the community |
```
