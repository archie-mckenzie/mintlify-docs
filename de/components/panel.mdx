---
title: "Panel"
description: "Legen Sie den Inhalt des rechten Seitenpanels fest"
icon: "panel-right"
---

Sie können die Komponente `<Panel>` verwenden, um das rechte Seitenpanel einer Seite mit beliebigen Komponenten anzupassen.

Wenn eine Seite eine `<Panel>`-Komponente enthält, müssen die Komponenten [RequestExample](/de/components/examples#request-example) und [ResponseExample](/de/components/examples#response-example) innerhalb von `<Panel>` platziert werden.

Die Komponenten in einem `<Panel>` ersetzen das Inhaltsverzeichnis der Seite.

```mdx
<Panel>
  <Info>Heften Sie Informationen an das Seitenpanel an. Oder fügen Sie eine andere Komponente hinzu.</Info>
</Panel>
```
