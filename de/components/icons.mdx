Verwenden Sie Icons aus Font Awesome, Lucide, SVGs, externen URLs oder Dateien in Ihrem Projekt, um Ihre Dokumentation zu verbessern.

<Icon icon="flag" size={32} />

```mdx Icon example

```

<div id="inline-icons">
  ## Inline-Symbole
</div>

Symbole werden inline platziert, wenn sie innerhalb eines Absatzes verwendet werden.

<Icon icon="flag" iconType="solid" /> Die gewünschte Dokumentation, ganz ohne Aufwand.

```markdown Inline icon example
<Icon icon="flag" iconType="solid" /> The documentation you want, effortlessly.
```

<div id="properties">
  ## Eigenschaften
</div>

<IconsRequired />

<ResponseField name="color" type="string">
  Die Farbe des Icons als Hexcode (zum Beispiel `#FF5733`).
</ResponseField>

<ResponseField name="size" type="number">
  Die Größe des Icons in Pixel.
</ResponseField>
