Verwenden Sie die Komponente `CodeGroup`, um mehrere Codeblöcke in einer tab-basierten Oberfläche darzustellen. So können Benutzer Implementierungen in verschiedenen Programmiersprachen vergleichen oder alternative Ansätze für dieselbe Aufgabe betrachten.

<CodeGroup>

```javascript helloWorld.js
console.log("Hello World");
```

```python hello_world.py
print('Hello World!')
```

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

</CodeGroup>

<div id="creating-code-groups">
  ## Code-Gruppen erstellen
</div>

Um eine Code-Gruppe zu erstellen, fassen Sie mehrere Codeblöcke mit `<CodeGroup>`-Tags zusammen. Jeder Codeblock muss einen Titel enthalten, der als Tab-Beschriftung dient.

````mdx
<CodeGroup>

```javascript helloWorld.js
console.log("Hello World");
```

```python hello_world.py
print('Hello World!')
```

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

</CodeGroup>
````

<div id="language-dropdown">
  ## Sprachen-Dropdown
</div>

Sie können die Tabs in einer Codegruppe durch ein Dropdown-Menü ersetzen, um mit dem `dropdown`-Prop zwischen Sprachen umzuschalten.

<CodeGroup dropdown>


</CodeGroup>
