---
title: "Rollen"
description: "Steuern Sie den Zugriff auf Ihr Dashboard mithilfe von Rollen."
---

<Info>
  RBAC-Funktionalität ist im [Enterprise-Plan](https://mintlify.com/pricing?ref=rbac) verfügbar.
</Info>

Mintlify bietet zwei Zugriffslevel für das Dashboard: Editor und Admin.

Die folgenden Aktionen sind der Admin-Rolle vorbehalten:

|                               | Editor | Admin |
| ----------------------------- | :----: | :---: |
| Rollen von Benutzer:innen ändern |   ❌   |  ✅   |
| Benutzer:innen löschen        |   ❌   |  ✅   |
| Admins einladen               |   ❌   |  ✅   |
| Abrechnung verwalten und aktualisieren |   ❌   |  ✅   |
| Eigene Domain aktualisieren   |   ❌   |  ✅   |
| Git-Quelle aktualisieren      |   ❌   |  ✅   |
| Organisation löschen          |   ❌   |  ✅   |

Andere Aktionen im Dashboard stehen beiden Rollen zur Verfügung.

Sie können beliebig viele Admins einladen. Wir empfehlen jedoch, den Admin-Zugriff auf Personen zu beschränken, die ihn benötigen.
