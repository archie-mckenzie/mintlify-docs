Fügen Sie Ihrer Dokumentation Bilder hinzu, betten Sie Videos ein und binden Sie interaktive Inhalte per iframe ein.

Alle statischen Assets in Ihrem Docs-Repository werden automatisch unter dem entsprechenden Pfad auf Ihrer Domain ausgeliefert. Wenn Sie zum Beispiel `/images/my-logo.png` in Ihrem Repo haben, ist das Bild unter `https://docs.yoursite.com/images/my-logo.png` verfügbar.

<div id="images">
  ## Bilder
</div>

Fügen Sie Bilder hinzu, um visuellen Kontext, Beispiele oder Dekoration in Ihrer Dokumentation bereitzustellen.

<div id="basic-image-syntax">
  ### Grundlegende Bildsyntax
</div>

Verwenden Sie die [Markdown-Syntax](https://www.markdownguide.org/basic-syntax/#images), um Bilder zu Ihrer Dokumentation hinzuzufügen:

```mdx
![Alt-Text, der das Bild beschreibt](/pfad/zum/bild.png)
```

<Tip>
  Fügen Sie immer beschreibenden Alt-Text hinzu, um die Barrierefreiheit und SEO zu verbessern. Der Alt-Text sollte klar beschreiben, was das Bild zeigt.
</Tip>

Bilddateien müssen kleiner als 20 MB sein. Für größere Dateien hosten Sie diese auf einem CDN-Service wie [Amazon S3](https://aws.amazon.com/s3) oder [Cloudinary](https://cloudinary.com).

<div id="html-image-embeds">
  ### HTML-Bildeinbettungen
</div>

Für mehr Kontrolle über die Bilddarstellung verwenden Sie HTML-`<img>`-Tags:

```html
<img 
  src="/images/dashboard.png" 
  alt="Hauptdashboard-Oberfläche"
  height="300"
  className="rounded-lg"
/>
```

<div id="disable-zoom-functionality">
  #### Zoom-Funktionalität deaktivieren
</div>

Um die standardmäßige Zoom-Funktion beim Klicken für Bilder zu deaktivieren, fügen Sie die `noZoom`-Eigenschaft hinzu:

```html highlight="4"
<img 
  src="/images/screenshot.png" 
  alt="Beschreibender Alt-Text"
  noZoom 
  height="200"
/>
```

<div id="link-images">
  #### Bilder verlinken
</div>

Um ein Bild zu einem anklickbaren Link zu machen, umschließen Sie das Bild mit einem Anker-Tag und fügen Sie die `noZoom`-Eigenschaft hinzu:

```html
<a href="https://mintlify.com" target="_blank">
  <img 
    src="/images/logo.png" 
    alt="Mintlify-Logo"
    noZoom 
    height="100"
  />
</a>
```

<Note>
  Bilder innerhalb von Anker-Tags zeigen automatisch einen Zeiger-Cursor an, um zu signalisieren, dass sie anklickbar sind.
</Note>

<div id="light-and-dark-mode-images">
  #### Bilder für hellen und dunklen Modus
</div>

Um verschiedene Bilder für helle und dunkle Themes anzuzeigen, verwenden Sie Tailwind-CSS-Klassen:

```html
<!-- Bild für hellen Modus -->
<img 
  className="block dark:hidden" 
  src="/images/light-mode.png" 
  alt="Oberfläche im hellen Modus"
/>

<!-- Bild für dunklen Modus -->
<img 
  className="hidden dark:block" 
  src="/images/dark-mode.png" 
  alt="Oberfläche im dunklen Modus"
/>
```

<div id="videos">
  ## Videos
</div>

Mintlify unterstützt [HTML-Tags in Markdown](https://www.markdownguide.org/basic-syntax/#html) und bietet damit die Flexibilität, reichhaltige Inhalte zu erstellen.

<Tip>
  Fügen Sie immer Fallback-Text in Videoelemente ein, für Browser, die keine Videowiedergabe unterstützen.
</Tip>

<div id="youtube-embeds">
  ### YouTube-Einbettungen
</div>

Betten Sie YouTube-Videos über iframe-Elemente ein:

```html
<iframe
  className="w-full aspect-video rounded-xl"
  src="https://www.youtube.com/embed/4KzFe50RQkQ"
  title="YouTube video player"
  frameBorder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowFullScreen
></iframe>
```

</Frame>

<div id="self-hosted-videos">
  ### Selbst gehostete Videos
</div>

Verwenden Sie das HTML-Element `<video>` für selbst gehostete Videoinhalte:

```html
<video
  controls
  className="w-full aspect-video rounded-xl"
  src="link-to-your-video.com"
></video>
```

<div id="autoplay-videos">
  ### Automatisch abspielende Videos
</div>

Um ein Video automatisch abzuspielen, verwenden Sie:

```html
<video
  autoPlay
  muted
  loop
  playsInline
  className="w-full aspect-video rounded-xl"
  src="/videos/demo.mp4"
></video>
```

<Note>
  Bei Verwendung der JSX-Syntax schreiben Sie mehrteilige Attribute in camelCase: `autoPlay`, `playsInline`, `allowFullScreen`.
</Note>

<div id="iframes">
  ## iframes
</div>

Externe Inhalte mit iframe-Elementen einbinden:

```html
<iframe 
  src="https://example.com/embed" 
  title="Embedded content"
  className="w-full h-96 rounded-xl"
></iframe>
```
