Versteckte Seiten werden aus der Navigation Ihrer Website entfernt, bleiben jedoch für alle öffentlich zugänglich, die ihre URL kennen.

Verwenden Sie versteckte Seiten für Inhalte, die auf Ihrer Website zugänglich sein oder als Kontext für KI-Tools dienen sollen, aber nicht über die Navigation auffindbar sein sollen.

Für Inhalte, die eine strikte Zugriffskontrolle erfordern, müssen Sie die [Authentifizierung](/de/authentication-personalization/authentication-setup) konfigurieren.

Wenn Sie Seiten für bestimmte Nutzergruppen ausblenden möchten, verwenden Sie die Personalisierung, um die [Sichtbarkeit von Seiten](/de/authentication-personalization/overview#page-visibility) zu steuern.

<div id="hiding-a-page">
  ## Eine Seite ausblenden
</div>

Eine Seite ist ausgeblendet, wenn sie nicht in Ihrer `docs.json`-Navigation enthalten ist. Um eine Seite auszublenden, entfernen Sie sie aus Ihrer Navigationsstruktur.

Ausgeblendete Seiten verwenden dieselbe URL-Struktur wie reguläre Seiten, basierend auf ihrem Dateipfad. Zum Beispiel wäre `guides/hidden-page.mdx` unter `docs.yoursite.com/guides/hidden-page` erreichbar.

Siehe ein [Beispiel für eine ausgeblendete Seite](/de/guides/hidden-page-example).

<Note>
  Einige Navigationselemente wie Seitenleisten, Dropdowns und Tabs können auf ausgeblendeten Seiten leer erscheinen oder das Layout verschieben.
</Note>

<div id="hiding-a-group-of-pages">
  ## Eine Gruppe von Seiten ausblenden
</div>

Eine Seitengruppe wird ausgeblendet, wenn die Eigenschaft `hidden` in Ihrer `docs.json` auf `true` gesetzt ist:

```json highlight={4}
"groups": [
  {
    "group": "Getting started",
    "hidden": true,
    "pages": [
      "index",
      "quickstart"
    ]
  },
  {
    "group": "Guides",
    "pages": [
      "guides/hidden-page.mdx",
      "guides/hidden-groups.mdx"
    ]
  }
]
```

In diesem Beispiel ist die Gruppe `Getting started` ausgeblendet, die Gruppe `Guides` hingegen sichtbar.

<div id="hiding-a-tab">
  ### Einen Tab ausblenden
</div>

Sie können auch einen Tab ausblenden, indem Sie die Eigenschaft `hidden` in Ihrer `docs.json` hinzufügen:

```json highlight={4}
"tabs": [
  {
    "tab": "Home",
    "hidden": true,
    "pages": [
      "index",
      "quickstart"
    ]
  }
]
```

<div id="search-seo-and-ai-indexing">
  ## Suche, SEO und KI-Indexierung
</div>

Standardmäßig sind versteckte Seiten von der Indexierung für Suchmaschinen, die interne Suche in Ihrer Dokumentation und als Kontext für den KI-Assistenten ausgeschlossen. Um versteckte Seiten in Suchergebnissen und als Kontext für den Assistenten einzubeziehen, fügen Sie die Eigenschaft `seo` zu Ihrer `docs.json` hinzu:

```json
"seo": {
    "indexing": "all"
}
```

Um eine bestimmte Seite auszuschließen, fügen Sie `noindex: "true"` zu deren Frontmatter hinzu.
